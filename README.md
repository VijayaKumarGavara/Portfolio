# Personal Portfolio Website

A responsive personal portfolio website built to showcase my projects, skills, and experience, with a focus on clean UI, performance, and real-world best practices.

Live site: https://vijayakumargavara-1dbfb.web.app

---

## Overview

This portfolio is designed as a single-page React application that highlights my work, technical skills, and experience in a simple and accessible way.  
The project was built with an emphasis on:

- Performance on mobile and desktop
- Clean and readable UI
- SEO basics and accessibility
- Maintainable and scalable code structure

It also includes a small dynamic feature that fetches the latest GitHub commit date to reflect active development.

---

## Features

- Responsive layout optimized for mobile and desktop
- Dark mode support
- Section-based navigation (About, Skills, Projects, Experience, Contact)
- Lazy-loaded GitHub last commit indicator
- SEO-friendly structure and metadata
- Optimized fonts and assets for performance
- Hosted using Firebase Hosting

---

## Tech Stack

**Frontend**
- React (Vite)
- Tailwind CSS

**Performance & Optimization**
- Code splitting and lazy loading
- Font optimization
- Lighthouse-based performance tuning

**Hosting**
- Firebase Hosting
